**Rodar onde quero instalar, ele vai criar uma pasta com o nome do app, 
que posso mudar substituindo o "app-intro"**

**Cria um novo projeto React
npx create-react-app app-intro2

**Inicia o app React
cd dev
cd app-intro
npm start

**Instala bootstrap**
npm install bootstrap

**Instala dotenv**
npm install dotenv --save

**Instala Axios**
npm install axios --save

**Instala dependencias do PrimeReact**
npm install primereact --save
npm install primeicons --save
npm install react-transition-group --save

**Instala react-router-dom**
npm install --save react-router-dom

** Instala React Hook Form**
npm install react-hook-form --save
